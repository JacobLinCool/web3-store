# web3-store

A CLI Tool to Store Files on Web3 Storage (IPFS) with Encryption.

## Features

-   [x] Store Files on web3.storage (IPFS)
    -   [x] With Encryption
    -   [x] Without Encryption
-   [ ] Retrieve Files from web3.storage (IPFS)
    -   [ ] Encrypted Files
    -   [ ] Unencrypted Files
-   [ ] Encrypt File Locally
-   [x] Decrypt File Locally
